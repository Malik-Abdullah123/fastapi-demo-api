# Insurance Premium Category Predictor

This is a simple web app to predict the **insurance premium category** based on user inputs using **FastAPI** (backend) and **Streamlit** (frontend).

## Features

- Input personal details: age, weight, height, annual income, smoker status, city, occupation.
- Predict insurance premium category: **Low, Medium, High**.
- Built with **FastAPI** for backend and **Streamlit** for frontend.
